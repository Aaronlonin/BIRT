How to use a template report item

To use a template report item, you must create a report design from a template that contains a template report item. This example uses the predefined Chart and Listing template.

1
Create a report design. In the New Report wizard, choose the Chart and Listing predefined template.
2
In the layout editor, follow the instructions in the cheat sheet.
The final instruction is to edit the chart. The chart item is a template report item. This item appears in the layout with an appropriate icon and instructional text, as shown in Figure 19‑11. The text supplements the instructions in the cheat sheet. In this case, the instructions are similar.

Figure 19‑11
Using a template report item
When you double-click the template report item, the chart builder appears.
3
Set up the chart as described earlier in this online help. The chart now appears as a standard chart element in the layout.
4
To discard the changes that you made to the chart, right-click the chart element, and choose Revert to Template Item. The chart now appears as the original template report item, as shown in Figure 19‑11.
