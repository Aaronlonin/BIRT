How to specify a header or footer size

1
In the layout editor, choose the Master Page tab. The layout editor displays the master page.
2
In Property Editor, choose General properties. Property Editor displays the master page’s general properties, as shown in Figure 18‑1.
3
Specify a size for the header height, the footer height, or both.
