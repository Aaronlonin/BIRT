How to drop a theme from a report design

You drop a theme from a report design by changing the Theme property of the report root item.

1
To access the report design’s properties, in the Outline view, select the report root item.
Figure 19‑3 shows this item selected. This item shows either the report design’s file name or its title property value, if it has one.

Figure 19‑3
Report root item
2
In Property Editor, select None from the list in Themes, as shown in Figure 19‑4.

Figure 19‑4
Selecting a theme in Property Editor
Now, the theme that the report design uses is None, which means that it uses only its own styles or styles from an included CSS file, not styles from a theme in a library.
