Presenting Data in a Cross Tab

A cross tab displays data in a row-and-column matrix that looks similar to a spreadsheet. Like a spreadsheet, the cross tab is ideal for summarizing data in a compact and concise format. It displays summary, or aggregate, values such as sums, counts, or averages. The cross tab groups these values by one set of data listed down the left side of the matrix and another set of data listed across the top of the matrix.
